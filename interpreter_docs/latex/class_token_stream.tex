\doxysubsection{Detailed Description}
A class to handle the tokens generated by the lexer. 

\begin{DoxyNote}{Note}
This class is used to store the tokens generated by the lexer and to split them into lines for easier parsing, and assigning them to each \doxylink{struct_token}{Token} for easier error reporting. 
\end{DoxyNote}
